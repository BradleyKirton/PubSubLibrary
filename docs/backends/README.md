# PubSub Backends

> The following are supported as backends for handling the pub/sub

## Redis

The following environment variables will be used:

| Config        | Requred       | Default  |
| ------------- |:-------------:| -----:|
| `PUBSUB_HOST`       | no | 'redis' |
| `PUBSUB_PASSWORD`   | no | `None` |
| `PUBSUB_PORT`       | no | `6379` |
| `PUBSUB_INDEX`      | no | `0` |
